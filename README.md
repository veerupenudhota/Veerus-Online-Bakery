Online Bakery Management System

Project Description
The Online Bakery Management System is a web-based application designed to streamline the process of ordering bakery items online. The project is built using Spring Boot, Spring MVC, HTML, CSS, JavaScript, Java, and MySQL, ensuring a seamless and efficient user experience. Customers can browse available bakery products, add them to their cart, place orders, and make payments. Admins can manage inventory, process orders, and analyze sales data.

Key Features
✅ User Authentication & Authorization – Secure login and signup functionality for customers and admins.
✅ Product Management – View, add, update, and remove bakery items from the catalog.
✅ Shopping Cart & Checkout – Users can add items to the cart and proceed with an order.
✅ Order Management – Track and manage order status (Pending, Processing, Delivered).
✅ Payment Integration – Support for online payments and cash on delivery.
✅ Admin Dashboard – Manage users, orders, and inventory efficiently.
✅ Responsive Design – Fully functional across devices with a user-friendly UI.

Technology Stack:-
Backend: Spring Boot, Spring MVC, Java
Frontend: HTML, CSS, JavaScript
Database: MySQL
Frameworks & Libraries: Hibernate (ORM), Bootstrap (UI Design)
Tools: IntelliJ/Eclipse, MySQL Workbench, Postman, GitHub
